# Chatbot
## 동국대학교 주변 맛집 추천 챗봇

frontend : react, javascript  
api : flask (python)  
chatbot : tensorflow   (있는거 뭐잇냐)


## How to run

### 1. frontend (react) 실행
```
$ yarn install     
$ yarn start
```
### 2. chatbot 실행
```
$ conda activate chatbot  
$ python bot.py`  
$ cd ../../chatbot_api  
$ flask run 
```
