## How to run

### 1. frontend (react) 실행
```
$ yarn install     
$ yarn start
```
### 2. chatbot 실행
```
$ conda activate chatbot  
$ python bot.py  
```
### 3. api 실행
```
$ cd ../../chatbot_api  
$ flask run 
```
