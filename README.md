# Chatbot
- 동국대학교 주변 맛집 추천 챗봇

## Environment

frontend - react, javascript
api - flask (python)
chatbot - keras (python(tensorflow))

--> requirement.txt 에 정리하기

## How to run

### 1. chatbot 실행
```
$ conda activate chatbot  
$ python bot.py  
```
### 2. api 실행
```
$ cd ../../chatbot_api  
$ flask run 
```
### 3. frontend (react) 실행
```
$ yarn install     
$ yarn start
```
### 3. frontend (javascript) 실행
```
chatbot.html 실행
```
